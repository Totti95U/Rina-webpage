なぜエロスパムのような動画タイトル、サムネイルなんですか？
まず、私はすべあな界隈の本質が『擬態』であると考えています。例えば、本家すべあなはニコニコ動画の「削除動画」に擬態していると考えています。他の模倣においても、削除動画をはじめとしたエラー画面に擬態していると言えるものが多くあります。そもそも「模倣」という行為自体が本家への擬態と考えることができます。また、エラー画面以外にも例えば全てを騙る時雨に。ではゆっくり実況動画に擬態した動画が投稿されています。同じようにりなでも『擬態』という芯は残しつつも、エラー画面とは違うものに擬態したものにしようと考えました。そこで私は次の理由からYouTube上のコメント欄で稀に見られるエロスパム垢を擬態先に選びました：
  • エロスパム垢がbotであることからくる無機質さがエラー画面のもつそれと非常に似ていると感じた
  • りなは実験的作品、本垢に投稿するほどじゃない作品を投稿する場にしたかったので、おふざけをしても許される雰囲気を作りたかった
  • エロスパム垢風のタイトル、サムネイルは非常に目を引くものが多く、界隈外からの意図しない流入が増えることを期待した。界隈外からの意図しない流入は『擬態』の真骨頂だと考える。
  • 現在の王道的模倣スタイルから外れることで知る人ぞ知るチャンネルにしたかった
意外と真面目な理由だったでしょ？
激エロremix ってなんなんですか？私も激エロremixしていいですか？
私もよくわかっていませんが、CXXXIIから派生した全て貴方の所為です。の編曲に私の好きなものをねじ込んだものだと思っています。MVが3DCGなのは「いい感じにBlenderの練習になりそうだな～」と·[short] - 激エロ remix feat. 結月ゆかりを作りながら思ったからで、とくに深い意味はないです。「激エロ」もりな名義で上げたから以上の深い意味はありません。選曲についても、いまはいわゆる界隈曲 に近い雰囲気のものが多いですが、今後界隈曲外のものも作る予定があります。後半の質問についてですが、あなたの思う激エロremixがあれば是非作ってください。どんなエロが待っているのか私も聴いてみたいので。
